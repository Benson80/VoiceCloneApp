## 安装步骤

### 1. 克隆仓库

首先，将 `VoiceCloneApp` 项目克隆到本地：

```
git clone https://github.com/Benson80/VoiceCloneApp.git
```

### 2. 进入项目目录

克隆完成后，进入项目的根目录：

```
cd VoiceCloneApp
```

### 3. 安装依赖

使用 `pip3` 安装项目所需的依赖包。确保你已经安装了 `Python 3` 和 `pip`，然后执行以下命令：

```
pip3 install -r requirements.txt
```

### 4. 启动应用

安装完所有依赖后，运行应用：

```
python3 app.py
```

应用将会在本地启动，默认监听 `http://127.0.0.1:5000`。
